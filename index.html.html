🚀 Hosting guide

Put this app on GitHub Pages — free, permanent URL, works on any phone, no account needed to view.

1
Create a free GitHub account
Go to github.com, sign up (it's free). Confirm your email.
2
Create a new repository
Click + → New repository. Name it djubi-wc2026. Set to Public. Tick "Add a README file". Click Create.
3
Upload this file
In your new repo, click Add file → Upload files. Drag the worldcup-bet.html file you downloaded from Claude. Commit it. Then rename it to index.html (click the file, pencil icon, rename).
4
Enable GitHub Pages
Go to repo Settings → Pages. Under "Source" select Deploy from branch, branch main, folder / (root). Save.
5
Your live URL
After ~2 minutes your app is live at:
https://YOUR-USERNAME.github.io/djubi-wc2026/
Send this URL to Djubi S — no install needed, just open in any browser.
6
Saving your bets across phones
Both of you open the same URL and enter the same Room code (e.g. DJUBI26). The share link encodes the current state — whenever one of you updates, export the link and share it again. For true real-time sync, see the Netlify + Supabase upgrade path (ask Claude!).
